# Лабораторная работа: Система управления транспортной компанией

---

## Задание 1: Класс `Client`

Создать класс `Client` для представления клиентов компании.

**Атрибуты:**
- `name` — имя клиента (строка).
- `cargo_weight` — вес груза (число с плавающей точкой).
- `is_vip` — флаг VIP-статуса (логическое значение, по умолчанию `False`).

---

## Задание 2: Базовый класс `Vehicle`

Создать базовый класс `Vehicle` для представления транспортного средства.

**Атрибуты:**
- `vehicle_id` — уникальный идентификатор (строка, генерируется случайно при создании).
- `capacity` — грузоподъемность (число с плавающей точкой, в тоннах).
- `current_load` — текущая загрузка (число с плавающей точкой, по умолчанию `0`).
- `clients_list` — список клиентов, чьи грузы загружены (список объектов `Client`).

**Методы:**
- `load_cargo(client)` — увеличивает текущую загрузку на `cargo_weight` из объекта `client` и добавляет клиента в `clients_list`. Перед загрузкой проверить, что грузоподъемность не будет превышена, и выполнить валидацию типов данных.
- `__str__` — возвращает строку с информацией о транспортном средстве: ID, грузоподъемность и текущая загрузка.

---

## Задание 3: Наследование и класс `TransportCompany`

### 1. Класс `Airplane`
Создать класс `Airplane`, наследующий от `Vehicle`.

**Дополнительные атрибуты:**
- `max_altitude` — максимальная высота полета (число с плавающей точкой, в метрах).

---

### 2. Класс `Van`
Создать класс `Van`, наследующий от `Vehicle`.

**Дополнительные атрибуты:**
- `is_refrigerated` — флаг, указывающий на наличие холодильника (логическое значение, `True`/`False`).

---

### 3. Класс `TransportCompany`
Создать класс `TransportCompany` для управления транспортными средствами и клиентами.

**Атрибуты:**
- `name` — название компании (строка).
- `vehicles` — список транспортных средств (список объектов `Vehicle` и его наследников).
- `clients` — список клиентов (список объектов `Client`).

**Методы:**
- `add_vehicle(vehicle)` — добавляет транспортное средство в список `vehicles` (с валидацией типа).
- `list_vehicles()` — возвращает список всех транспортных средств.
- `add_client(client)` — добавляет клиента в список `clients`.
- `optimize_cargo_distribution()` — распределяет грузы клиентов по транспортным средствам:
  - Грузы VIP-клиентов загружаются в первую очередь.
  - Используется как можно меньше транспорта.

---

## Задание 4: Консольное меню

В файле `main.py` разработать консольное меню для удобной работы с клиентами, транспортными средствами и возможностью вывода результата распределения грузов.

**Требования к меню:**
- Добавление клиентов.
- Добавление транспортных средств.
- Распределение грузов.
- Вывод информации о транспортных средствах и клиентах.
- Выход из программы.
